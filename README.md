### 构建

#### 构建稳定版

从release下载code，并解压

```bash
yarn
yarn build
```

使用npm

```bash
npm -i
npm run build
```



启动

```bash
yarn start
```

使用npm

```bash
npm run start
```



#### 构建开发版

克隆仓库

```bash
git clone https://github.com/alongw/nia-qq-bot.git
```

进入目录

```
cd ./nia-qq-bot
```

打包

```bash
yarn
yarn build
```

或使用npm

```bash
npm -i
npm run build
```

启动

```bash
yarn start
```

或使用npm

```bash
npm run start
```
